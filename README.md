## 🚀 Quick Start

1. **Install Git**: Follow the [installation guide](Git%20Commands/0.Git%20Installation.md)
2. **Learn the basics**: Start with [Git Fundamentals](Git%20Commands/1.%20Git%20Fundamentals.md)
3. **Set up your first repo**: Use the [Repository Setup](Git%20Commands/2.%20Repository%20Setup.md) guide
4. **Configure Git**: Set up your identity with [Git Configuration](Git%20Commands/3.%20Git%20Configuration.md)
5. **Master daily commands**: Practice with [Daily Commands](Git%20Commands/Daily%20Commands.md)

## 📖 How to Use This Guide

- **Beginners**: Follow the guides in order, starting from installation
- **Intermediate users**: Jump to specific topics like branching or remote repositories
- **Advanced users**: Explore automation hooks, submodules, and advanced workflows
- **Reference**: Use any guide as a quick command reference

## 🛠️ What You'll Learn

- **Git Basics**: Understanding version control, repositories, and commits
- **Branching**: Creating, merging, and managing parallel development
- **Collaboration**: Working with remote repositories and team workflows
- **Advanced Features**: Hooks, stashing, tagging, and submodules
- **Best Practices**: Clean commit history, proper branching strategies
- **Troubleshooting**: Reverting changes, resolving conflicts, and recovery
